Getting started


As of 04/09/2020 our entire API is REST based.. If you want to do more reading on REST api's view here.

Various languages like Javascript and Python natively support connecting to REST API's

API Endpoints

GET /api/get/bot/:id

Get's the bots (provided the ID and ONLY if bot is listed) statistics, returns data from the Bot Model

GET /api/get/user/:id

Get's the users (provided the ID) statistics, returns data from the User Model

GET /api/protected/users/voted/:id

Get's if a user has voted for your bot, returns false or true depending on if they voted in the past 24 hours

Trouble Shooting

Botrix uses a Custom Error schema which is provided via the Botrix API. If you find yourself on an error page, don’t worry just copy the link to the page and reference this docs page for explanations on the error.

ERROR | /error/?e={error}

See below on error page schema

Error Error Desc
e=unknown provided ID was invalid.
e=verified That bot was already found on the botlist.
e=unverified That bot was already found on the botlist. Wait for verification.
e=long The short description was too long.
e=user Please logout and login again.
e=invite Invalid bot invite link provided.
e=owner The bot is not yours to edit.
e=id You can’t change the bot id.
e=server You are not in the server, Please join our Server before adding any bots.
e=js JS is not supported in the long description.
e=html HTML is not supported in the long description.
e=notfound Your bot was not found on the list.
e=nolegal Our Terms of Service and Privacy Policy are currently a work in progress, Please check back at a later date.
