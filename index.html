Botrix API

Botrix has its own API! It is freely available to download using:

npm install botrix-npm

Check the docs for more information on how to use the API.

Webhooks

Want to get a message every time someone votes for your bot? Now you can using webhook URL's like the one below:

https://discordapp.com/api/webhooks/[guild id]/[token]

Check our article on how to use this new feature.
